# Investigation App 🕵️

An interactive detective investigation web app where users solve mystery crime cases by investigating suspects, analyzing clues, interrogating characters, and identifying the criminal.

## Case: The Missing Diamond 💎

A priceless diamond disappeared from a mansion at 9 PM.  
Three suspects remain inside the mansion.

Your job is to:
- investigate suspects
- inspect clues
- analyze evidence
- interrogate characters
- identify the criminal
